# Mountain Recommendation System (SAW Method)
Sistem Pendukung Keputusan pemilihan gunung di Jawa Tengah.

## Fitur Utama
- Implementasi algoritma Simple Additive Weighting (SAW).
- Perhitungan normalisasi dan nilai preferensi (Vi) secara otomatis.
- Manajemen data kriteria dan alternatif gunung.

## Tech Stack
- PHP/Python (sesuaikan kodenya)
- MySQL Database
